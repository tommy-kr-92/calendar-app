# 일정 프로그램

---

## 환경

---

- Java 17
- Gradle
- Open-CSV

## 개요

---

### Create

- 일정 생성

### Read

- 개별 일정 생성
- CSV 파일 업로드를 통한 대량 일정 추가

### Update

- 일정 업데이트
- 삭제 유무 검증에 따른 업데이트 여부

### Delete 

- hard-delete가 아닌 soft-delete
- deleteYn(삭제 유무) 컬럼을 두어 삭제 여부 컨트롤
